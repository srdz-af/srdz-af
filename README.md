## Selected projects

### tsnl

C++ Graph Neural Network system for relation prediction between real-world entities.

I designed and implemented a from-scratch GraphSAGE pipeline with CSR graph storage, 2-layer inductive inference, minibatch sampling, and dense reindexing. The system scales to 170M nodes and 1.1B edges, reduces per-batch memory usage by 85%, and runs 100× faster than a naive full-neighborhood CPU baseline.

Repo: https://github.com/srdz-af/tsnl

### ND-Viewer

Interactive N-dimensional geometry viewer built with Three.js and TypeScript.

It supports n-dimensional objects, 3D projection, PCA projection, W rotation, vertex editing, JSON import/export, and undo/redo.

Repo: https://github.com/srdz-af/nd-viewer
